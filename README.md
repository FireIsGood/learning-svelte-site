# Learning Svelte Site

Made with [Astro](https://astro.build) so I could learn Svelte

## To run

```bash
npm run dev
```

Runs on port 3000

## To build

```bash
npm run build
```

Creates files to `/docs` folder for github pages.

NOTE: default base is `/learning-svelte-site` as configured in `astro.config.mjs`
